# 屏幕翻译 v1.1

Android 悬浮屏翻译 MVP：悬浮窗 + 屏幕选区 + 实时 OCR（ML Kit 日语/拉丁）+ DeepSeek 日英译中与日语语法解析。

## 本次更新（v1.1）
- **液态玻璃 UI**：悬浮窗与 App 内页面全面采用半透明渐变 + 描边 + 阴影的玻璃质感，全 API 级别兼容。
- **缩小悬浮窗**：紧凑胶囊控制条，更少遮挡。
- **加载态**：点刷新或触发新翻译时，结果卡片自动折叠，悬浮窗显示转圈加载图标；翻译完成后自动展开。
- **流畅过渡动画**：展开/折叠用弹性 spring 动画，箭头图标平滑旋转，加载↔结果切换连续无跳变。
- **手动刷新**：悬浮窗与主屏新增刷新入口，绕过去重缓存强制重译。
- **可读错误提示**：未配置 Key、401/402/429/5xx、坏 JSON 等均显示中文可理解错误。

## 已知限制（MVP）
- 选区固定为屏幕上半幅，暂未做拖拽选框交互。
- 仅 Wi-Fi 开关已持久化，尚未在请求链路真正拦截。
- 仅 debug 构建（未签名），仅供本机测试安装。

## 使用
1. 安装 APK，授予悬浮窗权限。
2. 在 App 设置中填写 DeepSeek API Key。
3. 点「开始翻译服务」，授权屏幕捕获，悬浮窗出现。
4. 自动识别选区文字并翻译，点悬浮窗刷新可重译。

> ⚠️ 这是 debug APK，未做正式签名，安装时可能需要允许「未知来源」。
