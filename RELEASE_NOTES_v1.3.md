# 屏幕翻译 v1.3

本次聚焦**悬浮球交互重构**：折叠态改为单独小球，刷新时自动折叠并在球上显示加载动画，完成后自动展开。

## 本次更新（v1.3）
- **折叠态 = 单独小球**：服务启动后屏幕上只显示一个 56dp 深色霓虹圆球（青绿色描边 + 径向辉光），空闲时球心显示「译」字，可拖动。
- **点击小球只展开阅读面板**，不触发刷新。
- **长按小球关闭服务**（保留彻底退出能力）。
- **刷新流程自动化**：
  - 点「刷新」/触发新翻译时，阅读面板**自动折叠回小球**，球心「译」字变为旋转的青绿色加载圈。
  - 翻译完成后**面板自动展开**显示译文。
- **所有结束情况都自动展开**：成功、DeepSeek 失败、未配置 Key、空文本——均展开并显示对应内容或原因。
- 球↔面板切换用 `AnimatedVisibility` + scale/fade 动画平滑过渡，沿用深色霓虹 UI 风格。
- `CaptureService` 流程无需改动，其 `showLoading/updateText/showError` 调用直接驱动新的状态机。

## 已知限制（MVP）
- 选区固定为屏幕上半幅，暂未做拖拽选框交互。
- 仅 Wi-Fi 开关已持久化，尚未在请求链路真正拦截。
- 仅 debug 构建（未签名），仅供本机测试安装。

## 使用
1. 安装 APK，授予悬浮窗权限。
2. 在 App 设置中填写 DeepSeek API Key。
3. 点「开始翻译服务」，授权屏幕捕获，悬浮球出现。
4. 点击悬浮球打开阅读面板；点面板内「刷新」会自动折叠→加载→展开。

> ⚠️ 这是 debug APK，未做正式签名，安装时可能需要允许「未知来源」。
