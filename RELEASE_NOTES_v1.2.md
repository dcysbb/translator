# 屏幕翻译 v1.2

本次为 **UI 全面重构**：从浅色玻璃风格切换到深色霓虹（teal/cyan/purple）主题，主屏、设置页、悬浮窗全部重绘。

## 本次更新（v1.2）
- **深色霓虹主题**：深色渐变背景 + 青/蓝/紫强调色 + 玻璃质感卡片，夜间观感更佳。
- **主屏重绘**：
  - 脉冲光晕图标（sweepGradient 环形光效 + 径向辉光）。
  - 系统状态卡，状态点呼吸动画。
  - 主/次级按钮带按压回弹（scale）动画。
- **设置页重绘**：分组式卡片、深色输入框、API Key 可见性切换（眼睛图标）。
- **页内导航**：主屏↔设置用 `AnimatedContent` 横向滑动 + 淡入淡出过渡。
- **悬浮窗**：深色胶囊控制条 + 底部强调色指示条；加载态改为脉冲圆点。
- **代码清理**：返回箭头改用 `AutoMirrored` 版本（RTL 正确）。

## 已知限制（MVP）
- 选区固定为屏幕上半幅，暂未做拖拽选框交互。
- 仅 Wi-Fi 开关已持久化，尚未在请求链路真正拦截。
- 仅 debug 构建（未签名），仅供本机测试安装。

## 使用
1. 安装 APK，授予悬浮窗权限。
2. 在 App 设置中填写 DeepSeek API Key。
3. 点「开始翻译服务」，授权屏幕捕获，悬浮窗出现。
4. 自动识别选区文字并翻译，点悬浮窗刷新可重译。

> ⚠️ 这是 debug APK，未做正式签名，安装时可能需要允许「未知来源」。
